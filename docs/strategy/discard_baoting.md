# 保听策略 - 在查叫/退税压力下优先成听

## 核心原则

**不听/花猪的流局惩罚是巨大负EV**。很多局的最优解是"先听保命，再择机加番"。

## 触发条件

1. 牌墙剩余<30张 且 你未听牌 且 向听≥2 → 高压力
2. 已有1-2家胡牌 且 牌局可能提前进入少人对局 → 中压力
3. 手中仍有缺门牌未清完 且 牌墙剩余<40 → 花猪风险

## 决策规则

1. IF 花猪风险(手中仍有缺门牌+牌墙不多) THEN **最高优先级**清缺门 — 花猪惩罚远超一切
2. IF 查叫压力高(牌墙<30且未听) THEN 牺牲番型也要先成听 — 听牌的EV显著高于追番
3. IF 一向听且有多种成听路线 THEN 选进张最多的成听路线，而非番数最高的
4. IF 已听牌但听口极窄(单钓/已见3张) THEN 考虑换听到更宽的听口

## 保听 vs 追番的EV比较

设 $P_{listen}$ = 成听概率，$Penalty$ = 查叫赔付(通常等于对方最大胡牌得分)

$$EV_{保听} = P_{listen} \times (避免Penalty + 可能胡牌收益)$$
$$EV_{追番} = P_{highFan} \times 高番收益 - (1 - P_{highFan}) \times Penalty$$

当 $Penalty$ 大(查叫体系)且 $P_{highFan}$ 不高时，保听几乎总是更优。

## 流局惩罚体系

| 身份 | 惩罚 |
|---|---|
| 花猪(缺门未清完) | 向每个非花猪未胡者赔付对方最大听牌得分 |
| 未听牌(非花猪) | 向已听牌者赔付其最大可能胡牌得分 |
| 未听牌+有杠收入 | 可能被要求退还全部杠钱(退税) |

## 常见误区

- ❌ 贪大番忽略流局风险 → ✅ 不听的负EV在查叫体系下极大
- ❌ "反正要流局了随便打" → ✅ 流局听牌 vs 不听的收益差距巨大
- ❌ 为保番型卡在二向听 → ✅ 降低番型但成听远优于高番型但未听
