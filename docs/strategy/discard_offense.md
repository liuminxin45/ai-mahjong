# 中盘进攻出牌 - 用向听和进张驱动每一次弃牌选择

## 核心决策流程（严格优先级）

1. **降向听优先**: IF 弃牌A能降向听而B不能 THEN 选A — 向听下降≈成听概率陡增
2. **同向听比进张**: IF 向听相同 THEN 选进张总张数更大的弃牌 — $p \approx R/N$，进张越多改善概率越高
3. **同进张比结构**: IF 进张也相同 THEN 保留复合形/四连/两面，弃低效形(边张/嵌张/孤张)
4. **避免有效牌重复**: 两个搭子共用相同进张时，拆掉其中一个
5. **保对子做雀头**: 无将(无对子)时，保留能自然形成雀头的结构

## 牌效价值排序（从高到低）

| 形状 | 进张种类 | 示例 | 价值 |
|---|---|---|---|
| 四连形 | 4种 | 3456听2/3/6/7 | 极高 |
| 两面+对子复合 | 3种 | 345+55 | 高 |
| 两面搭子 | 2种 | 45听3/6 | 高 |
| 嵌张 | 1种 | 46听5 | 中 |
| 边张 | 1种 | 12听3 | 低 |
| 孤张 | 0种(仅改良) | 单张8 | 最低 |

## 中张 vs 边张保留原则

顺子参与度: 1/9→1种, 2/8→2种, 3-7→3种

- 开局/中盘进攻态: 保留中张(3-7)，先弃边张幺九(1/9) → 再弃2/8 → 再弃3/7
- 保留5万比保留1万的期望进张多3倍

## 番型追求判断

| 条件 | 可追番型 | 额外价值 |
|---|---|---|
| 同色≥10张+牌型好 | 清一色(2番) | 翻4倍 |
| 刻子/对子≥4组 | 对对胡(2番) | 翻4倍 |
| 对子≥6对 | 七对子(2番) | 翻4倍 |
| 七对+含4张同牌 | 龙七对(3番) | 翻8倍 |

## 常见误区

- ❌ 只看"打什么安全"不看向听 → ✅ 进攻态优先降向听，防守是另一个模块
- ❌ 拆两面搭保孤张 → ✅ 两面搭进张2种(最多8张)远优于孤张
- ❌ 过早追高番放弃效率 → ✅ 先保证向听低、进张多，番型是锦上添花
- ❌ 有效牌重复不自知 → ✅ 两个搭子都听同一张时要拆一个
