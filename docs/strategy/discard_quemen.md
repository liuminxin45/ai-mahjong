# 缺门清理 - 在硬约束下最小化节奏损失

## 硬约束

手中有缺门牌时，**必须且只能打缺门牌**。违反此约束无法胡牌。

## 出牌顺序（缺门牌内部排序）

1. IF 缺门牌中有孤张幺九(1/9) THEN 先打 — 最低牌效，最安全
2. IF 缺门牌中有孤张边张(2/8) THEN 次打 — 低牌效
3. IF 缺门牌中有孤张中张(3-7) THEN 再打 — 注意: 中张可能对他人有用，有点炮风险
4. IF 缺门牌中有对子 THEN 可考虑先打一张保留一张(防止暴露)
5. IF 缺门牌中有搭子(如34) THEN 拆搭子打，先打外侧(3比4安全)

## 策略考量

- **速度**: 缺门越快清完，越早获得出牌自由度
- **安全**: 清缺门时打出的牌也可能点炮，注意场面信息
- **信息隐藏**: 不要因为清缺门暴露做牌方向(如连续打同一门的牌会暗示你在做另一门)

## 与碰/杠的交互

- 缺门花色**不能碰也不能杠** — 即使别人打出你缺门的牌
- 但你可以碰/杠**非缺门**花色(碰后仍需继续打缺门牌)

## 常见误区

- ❌ 缺门有牌时偷打别门 → ✅ 规则不允许，系统会阻止
- ❌ 不考虑清缺门的打牌顺序 → ✅ 合理排序可减少点炮风险
- ❌ 清缺门时随意打中张 → ✅ 4/5/6中张危险度更高，要结合场面判断
