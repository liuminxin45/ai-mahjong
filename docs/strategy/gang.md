# 杠决策 - 杠是投资决策，不是"有杠必杠"

## 杠收益与风险

| 杠类型 | 即时收益 | 抢杠风险 | 信息暴露 |
|---|---|---|---|
| 暗杠 | 向所有未胡玩家各收10分 | 一般不能被抢 | 低(不暴露具体牌) |
| 明杠(直杠) | 向点杠者收10分 | 不能被抢 | 高(暴露做牌方向) |
| 加杠(补杠) | 向原碰来源者收5分 | **可以被抢杠胡** | 高 |

## 杠EV公式

$$EV_{杠} = 即时杠钱 + 补牌价值 + 1番加成 - 抢杠风险 - 退税风险 - 信息暴露成本$$

## 决策规则（按优先级检查）

1. IF 加杠 且 有对手明显快听(多副露/快速清缺门) THEN **不杠** — 抢杠胡风险太高
2. IF 杠后向听数会上升 THEN **不杠** — 杠破坏手牌结构
3. IF 你未听牌 且 流局压力大 THEN **先听再杠** — 退税风险(流局不听退回全部杠钱)
4. IF 暗杠 且 不破坏手牌结构 THEN **杠** — 暗杠收益高、风险低
5. IF 明杠 且 杠后仍能维持听牌 THEN **杠** — 净收益为正
6. IF 加杠 且 场面安全(对手不像快听) THEN **杠** — 额外收益+补牌机会

## 杠后效应

- 杠后补1张牌 → 可能杠上开花(2番)
- 每个杠副露在计分时+1番 → 影响最终倍数
- 杠暴露信息 → 对手更容易判断你的做牌方向

## 退税规则

流局时:
- 未听牌者可能被要求退还全部杠钱收入
- 这意味着: 杠了但最后没听 = 净亏损

## 常见误区

- ❌ 有杠必杠 → ✅ 杠是投资决策，需要EV>0才值得
- ❌ 忽略抢杠胡风险 → ✅ 加杠时对手可以抢杠胡，视为放炮
- ❌ 不考虑退税 → ✅ 杠了但流局不听，杠钱要退回
- ❌ 杠后不调整策略 → ✅ 杠后手牌结构变了，需要重新评估向听和进张
