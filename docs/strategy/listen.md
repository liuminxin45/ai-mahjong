# 听牌选择 - 用EV比较不同听口

## 核心公式

对每个候选听牌方案 $j$:

$$EV_j = P_j \times Score_j - Risk_j - P(\text{流局不听}) \times Penalty$$

其中:
- $P_j$ = 在剩余摸牌次数内成胡的概率
- $Score_j$ = 胡牌得分(自摸vs点炮，番型倍数)
- $Risk_j$ = 为达到该听口需打出危险牌的放炮风险
- $Penalty$ = 查叫/花猪惩罚

## 听牌类型对照表

| 听牌类型 | 示例 | 理论最大胡张 | 评级 |
|---|---|---|---|
| 多面听/复合形 | 34567听2/5/8 | 11张 | 极强 |
| 两面 | 45听3/6 | 8张 | 好 |
| 双碰 | 44+99听4/9 | 8张 | 分情况(看剩余张) |
| 嵌张 | 46听5 | 4张 | 一般 |
| 边张 | 12听3 | 4张 | 偏差 |
| 单钓 | 等将牌 | 2张 | 差 |

**实际胡张数 = 理论最大 - 已见张数**，必须用可见信息修正。

## 决策规则

1. IF 两个听口EV差距>20% THEN 选高EV听口
2. IF EV接近 THEN 选进张更多的(自摸概率高)
3. IF 自摸加成大(规则倍率高) THEN 偏向宽听口(追自摸)
4. IF 某听口需要打出高危险牌 THEN 扣除放炮风险后再比较
5. IF 双碰听且其中一种牌已见3张 THEN 实际只有1张有效，视为死听

## 自摸 vs 点炮的收益差异

- 自摸: 每个未胡玩家**各付全额** → 收益 = Score × 未胡人数
- 点炮: 放炮者**独自付全额** → 收益 = Score × 1

血战到底中自摸加成越大，越应追求宽听口自摸路线。

## 常见误区

- ❌ 只看番数不看进张 → ✅ 4张听口×4倍 可能不如 8张听口×1倍
- ❌ 忽略"已见张数"修正 → ✅ 理论8张但已见6张=实际2张，比单钓还差
- ❌ 换听时不考虑打出牌的风险 → ✅ 换听打出的牌可能直接点炮
- ❌ 死守窄听等大番 → ✅ 血战中宽听快胡的总收益通常更高
