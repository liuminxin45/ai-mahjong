# 尾局 - 查叫/花猪惩罚驱动的终盘策略

## 尾局定义

牌墙剩余<20张 或 已有2家胡牌(仅剩2人对局)

## 最后四张规则

部分桌规: 最后四张有胡必胡(强制)。此规则存在时:
- 失去"见胡不胡等大番"的自由度
- 必须在进入最后阶段前完成听牌选择

## 流局惩罚体系（核心驱动力）

| 身份 | 惩罚 | 严重度 |
|---|---|---|
| 花猪 | 向每个非花猪未胡者赔付对方最大听牌得分 | 极高 |
| 未听牌(非花猪) | 向已听牌者赔付其最大可能胡牌得分 | 高 |
| 已听牌 | 无惩罚，可能从未听者获得赔付 | 收益 |

## 尾局决策优先级

1. **花猪检查**: IF 手中仍有缺门牌 THEN 不惜一切代价清完 — 花猪惩罚最重
2. **保听检查**: IF 未听牌 THEN 牺牲番型也要成听 — 查叫赔付远超追番收益
3. **退税检查**: IF 有杠收入且可能不听 THEN 评估是否会被退税
4. **安全出牌**: IF 已听牌 THEN 只打最安全的牌，等待自摸
5. **过水检查**: IF 跳过了胡牌 THEN 本圈不能再胡弃牌，需要等自摸或下圈

## 尾局EV逻辑

$$EV_{尾局} = P(自摸) \times Score - P(被点) \times Loss - P(流局) \times Penalty(身份)$$

当 $P(流局)$ 很高时，$Penalty(身份)$ 成为主导项:
- 花猪身份: Penalty最大 → 避免花猪是第一要务
- 未听身份: Penalty大 → 保听是第二要务
- 已听身份: Penalty=0且可能收益 → 坐等

## 常见误区

- ❌ 尾局还在贪大番 → ✅ 目标切换为"确保听牌+安全出牌"
- ❌ 忽略过水规则 → ✅ 尾局跳过胡牌的机会成本极高
- ❌ 不计算退税风险就杠 → ✅ 尾局杠了但没听=净亏损
- ❌ 认为流局无所谓 → ✅ 查叫/花猪惩罚可能比一次放炮更重
