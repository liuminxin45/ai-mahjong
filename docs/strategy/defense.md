# 防守切换 - 识别威胁信号并调整出牌安全度

## 防守触发条件

以下任一条件满足时，从进攻态切换到防守态:

1. 对手快速清缺门并停止丢某门 → 可能已听牌
2. 对手明副露≥2组(碰/杠频繁) → 手牌少、听牌概率高
3. 牌墙接近尾局(剩余<20张) → 危险牌密度上升
4. 已有1-2家胡牌(血战继续) → 放炮惩罚相对更重
5. 你自身向听≥3且牌墙<40 → 进攻无望，保听/保命

## 安全牌排序（从安全到危险）

1. **对手缺门花色的牌** → 其胡牌必须缺一门，该花色不可能胡
2. **场上已大量出现的牌(熟张)** → 剩余少，被听概率低
3. **幺九牌(1/9)** → 参与顺子方式少，相对安全
4. **2/8牌** → 参与顺子2种
5. **中张4/5/6** → **最危险**，参与顺子3种，信息缺失时风险最高

## 定缺信息利用

川麻定缺公开(或可强推断):
- 对手缺某门 → 该门牌对其安全
- 对手早早清空某门且做另一门 → 可能清一色，相关门牌极危险
- 对手大量打某门 → 该门大概率是其缺门

## 防守 vs 进攻的平衡

| 你的状态 | 推荐策略 |
|---|---|
| 已听牌+宽听口 | 维持进攻，自摸概率高 |
| 已听牌+窄听口 | 半攻半守，避免打高危险牌 |
| 一向听+进张多 | 偏进攻，但回避最危险牌 |
| 二向听+ | 偏防守，优先保听避惩罚 |
| 三向听+ | 全守，只打安全牌 |

## 血战特化

- 有人胡后放炮惩罚更重(只剩2-3人分摊风险)
- 已胡玩家不会再点炮 → 对其方向的牌变安全
- 后胡可能收益更高 → 但风险也更集中

## 常见误区

- ❌ 有人副露多就"全面防守" → ✅ 要结合你自身向听判断，已听牌可以继续进攻
- ❌ 只看牌面不看定缺信息 → ✅ 定缺信息是川麻最强的安全牌判断来源
- ❌ 防守时随意打牌 → ✅ 防守也有优先级，按安全牌排序打
- ❌ 忽略血战阶段变化 → ✅ 有人胡后局面风险结构会变
