# 定缺 - 选择清空成本最低且保留价值最高的两门

## 评分公式

给每门花色 $S$ 计算保留价值分(越高越值得保留):

$$Score(S) = 2 \times 连张数 + 3 \times 两面搭子数 + 2 \times 对子数 + 1 \times 中张数(3\text{-}7) - 2 \times 孤张幺九数$$

特化加分项:
- 若该门2/5/8集中度高(将对潜力) → +1~2分
- 若该门≥8张且多连张(清一色2番潜力) → +3分

**选 $Score$ 最低的门作为缺门。**

## 决策规则（按优先级排序）

1. IF 某门0张 THEN 直接缺该门(零成本)
2. IF 某门仅1-2张且全是孤张幺九 THEN 强烈倾向缺该门
3. IF 某门有刻子 THEN 避免缺该门(刻子难拆，且缺门不能碰杠)
4. IF 某门有顺子/两面搭 THEN 避免缺该门(进攻价值高)
5. IF 多门评分接近 THEN 选清空张数更少的门(减少被迫丢牌轮数)
6. IF 追清一色(某门≥10张且牌型好) THEN 可以缺掉"看似不多但会干扰主攻门"的花色

## 速查表

| 花色特征 | 适合缺 | 不适合缺 |
|---|---|---|
| 0-1张 | ✅ 极佳 | — |
| 2-3张全孤张 | ✅ 好 | — |
| 3-4张有对子 | ⚠️ 视情况 | — |
| 有刻子 | — | ❌ 刻子难拆且不能碰 |
| 有顺子/搭子 | — | ❌ 进攻价值高 |
| ≥5张多连张 | — | ❌ 做牌核心 |

## 硬约束提醒

- 定缺后: 有缺门牌时**只能打缺门牌**，不能打别门
- 定缺花色: **不能碰也不能杠**
- 胡牌前提: 手牌+副露中**不能有缺门花色**

## 常见误区

- ❌ 只看张数少就缺 → ✅ 3张有顺子的门可能比4张全孤张的门更值得保留
- ❌ 忽略清空成本 → ✅ 缺门需要消耗出牌轮数，影响做牌节奏
- ❌ 不考虑番型潜力 → ✅ 清一色2番、对对胡2番的潜力应影响定缺选择
