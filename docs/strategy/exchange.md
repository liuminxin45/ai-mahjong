# 换三张 - 最小化期望损失，制造两门不均衡

## 起手时序

- 发牌完成后，`P0` 先持 `14` 张，`P1-P3` 各持 `13` 张
- 在这个起手张数下进入换三张，不是四家都 `13` 张先换、再让 `P0` 补摸
- 换三张完成后再进入定缺，然后由 `P0` 直接开始首轮出牌

## 决策规则（按优先级排序）

1. IF 某门≥7张且质量高(多顺连/对子) THEN 考虑一门高番路线(清一色2番)，换出另一门最弱门
2. IF 三门较均衡 THEN 换出最差门，目标制造"两门结构"
3. IF 某门<3张 THEN 该门不够换出，把另一门最弱且≥3张的换出，少于3张的门作为缺门候选

## 选牌规则（从换出门中选3张）

**关键前提**: 换出的花色通常就是定缺花色。定缺 = 必须全部打掉、不能碰、不能杠、不能做将。
因此换出门剩余的牌全是废牌，对子/搭子在缺门中没有任何保留价值。

优先换出:
- 孤张幺九(1/9，无邻牌)
- 断张(与本门其他牌完全不搭)
- 会导致有效牌高度重复的牌

尽量不换出（仅针对保留门，不适用于缺门）:
- 成型对子(2张相同) — 但如果该门就是缺门，对子也应换出
- 连张(34/56等两面搭子)
- 四连形内部牌(如4567的5/6)

## ⚠️ 常见致命错误

- ❌ 换出3张后保留对子在缺门中，说"可以碰出或做将" → 定缺花色不能碰、不能做将！
- ❌ 用保留门的评估标准（保护对子）去评估缺门 → 缺门的牌全是废牌，越少越好

## 速查表

| 花色状态 | 换出优先级 | 原因 |
|---|---|---|
| 0-2张孤张 | 最高 — 直接作为缺门 | 清空成本最低 |
| 3-4张多孤张 | 高 — 优先换出 | 牌效价值低 |
| 5-6张有搭子 | 中 — 视整体判断 | 可能有发展潜力 |
| 7+张多连张 | 低 — 保留做主攻门 | 清一色/高效率候选 |

## 常见误区

- ❌ 只看张数少就换 → ✅ 还要看牌效价值(对子/刻子/顺子不应轻易换出)
- ❌ 换出高牌效牌追求"均衡" → ✅ 换三张目标是制造不均衡，不是均衡
- ❌ 忽略换入不确定性 → ✅ 换出是确定损失，换入是随机收益，底线是最小化损失
